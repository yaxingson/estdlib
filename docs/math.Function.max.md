# Function: max()

> **max**\<`T`\>(`array`): `void`

Computes the maximum value of array

## Type Parameters

• **T** *extends* `string` \| `number`

## Parameters

• **array**: `T`[]

The array to iterate over

## Returns

`void`

Returns the maximum value

## Since

0.1.2

## See

[source](https://github.com/yaxingson/estdlib/blob/main/lib/math/max.ts)

## Example

```js
max([4, 2, 8, 6]) // => 8

max([]) // => undefined

```
