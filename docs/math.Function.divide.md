# Function: divide()

> **divide**(`dividend`, `divisor`): `void`

Divide two numbers

## Parameters

• **dividend**: `number`

The first number in a division

• **divisor**: `number`

The second number in a division

## Returns

`void`

Returns the quotient

## Since

0.1.2

[source](https://github.com/yaxingson/estdlib/blob/main/lib/math/divide.ts)
